# 起起成长 · Good Luck Papa

> 一款帮助父母学会观察孩子的育儿助手 PWA，基于蒙台梭利教育理念设计。

**在线访问**：[https://goodluckpapa.github.io/GoodLuckPapa/](https://goodluckpapa.github.io/GoodLuckPapa/)

---

## 功能

- **今日推荐**：根据孩子月龄自动推荐活动，1 主 2 备，规则驱动
- **活动库**：0-12 月蒙台梭利活动（感官/运动/语言/情感/日常生活），含观察引导
- **观察记录**：4 字段极简记录（发起方式/专注时长/情绪/备注）
- **育儿日记**：结构化记录每日感想 + 图片，时间线按年/周折叠展示
- **成长里程碑**：5 维度里程碑对照，状态自动流转
- **兴趣识别**：F 规则本地计算，识别孩子的兴趣候选
- **周/月复盘**：5 问自检 + 蒙氏 10 维度月度评估
- **自定义活动**：创建家庭专属活动，自动纳入推荐
- **PWA 支持**：可安装到手机桌面，完全离线可用

## 技术栈

- 纯原生 HTML5 / CSS3 / Vanilla JS，无任何框架依赖
- IndexedDB 本地存储（全量数据保存在设备，刷新不丢失）
- Service Worker 离线缓存
- 适配手机端（375px 视口），可作为 PWA 安装

## 本地运行

```bash
npx serve . --listen 3000
# 访问 http://localhost:3000
```

## 数据安全

所有数据**仅存储在你的设备本地**（IndexedDB），不上传任何服务器。  
支持「设置 → 导出数据」将所有记录备份为 JSON 文件。
