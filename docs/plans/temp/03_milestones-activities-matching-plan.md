## 二、匹配规则说明

### 2.1 里程碑 ID 命名规范

```
m_{维度缩写}_{具体能力}

维度缩写映射：
- cognitive / cog      → cog
- motor                → motor
- language / lang      → lang
- emotional_social / emo → emotional
- self_care / self     → self
- social               → social
- math                 → math
- practical            → practical
- sensory / senso      → senso
- habit                → habit
```

### 2.2 维度定义

| 维度 | 说明 | 适用年龄 |
|------|------|---------|
| cognitive | 认知、思维、智力发展 | 0-72 月 |
| motor | 粗大动作和精细动作 | 0-72 月 |
| language | 语言理解和表达 | 0-72 月 |
| emotional_social | 情绪和社会性发展 | 0-72 月 |
| self_care | 自理能力和日常生活技能 | 12-72 月 |
| social | 社会交往和合作能力 | 18-72 月 |
| math | 数学心智和逻辑思维 | 24-72 月 |
| practical | 日常生活实践和优雅动作 | 24-72 月 |
| sensory | 感官体验和感知觉发展 | 0-72 月 |
| habit | 习惯养成和秩序感 | 0-72 月 |

### 2.3 月龄分段标准

| 月龄段 | 表示方式 | 示例 |
|--------|---------|------|
| 0-3 月 | 0-3 | windowStart: 0, windowEnd: 3 |
| 3-6 月 | 3-6 | windowStart: 3, windowEnd: 6 |
| 6-9 月 | 6-9 | windowStart: 6, windowEnd: 9 |
| 9-12 月 | 9-12 | windowStart: 9, windowEnd: 12 |
| 12-18 月 | 12-18 | windowStart: 12, windowEnd: 18 |
| 18-24 月 | 18-24 | windowStart: 18, windowEnd: 24 |
| 24-30 月 | 24-30 | windowStart: 24, windowEnd: 30 |
| ... | ... | ... |
| 69-72 月 | 69-72 | windowStart: 69, windowEnd: 72 |

### 2.4 活动-里程碑对应规则

1. **每个活动至少链接 1 个里程碑**，建议 2-3 个
2. **里程碑的月龄窗口必须覆盖活动的月龄范围**
3. **维度匹配**：活动的 domain 必须与里程碑的 dim 对应
4. **双向追溯**：每个里程碑需能反向找到关联的活动
