# 🚀 快速开始

三个不同角色的快速开始指南，选择适合你的：

---

## 👨‍👩‍👧 我是家长 — [for-parents.md](for-parents.md)
**想学会观察孩子，不需要技术背景**

⏱ **15 分钟快速上手**
1. 打开 `index.html`（无需安装）
2. 按照 App 内引导创建孩子档案
3. 开始今日推荐 → 观察 → 记录

**特点**：完全离线，数据保存在设备本地，无需联网

---

## 👨‍💻 我是开发者 — [for-developers.md](for-developers.md)
**想为项目贡献代码**

⏱ **20 分钟环境配置**
1. 安装 Node.js 和 npm
2. 运行 `npm install && npm test`
3. 修改代码，创建 Pull Request

**特点**：244 个单元测试全通过，开发友好的代码结构

---

## 🖥️ 我是部署者 — [for-deployment.md](for-deployment.md)
**想把数据同步到 NAS/服务器**

⏱ **5 分钟快速启动**
1. 有 Docker 和 Docker Compose
2. 运行 `docker-compose up -d`
3. 在 App 设置页配置 API 地址

**特点**：一键启动，支持小白部署，无需复杂配置
