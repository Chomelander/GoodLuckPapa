## 背景与目标

当前活动库仅覆盖 0-6 月（7 条）+ 6-9 月示范（3 条），共约 10 条活动。目标是将活动库扩充至完整覆盖 0-72 月（0-6 岁），总量约 150 条。

## 三阶段执行计划

```
阶段一：目录生成与确认
  · 按月龄段（3 月为一段，共 24 段）逐段输出活动目录
  · 每段格式：活动列表表格（ID / 名称 / 领域 / 核心概念 / 类型）+ 维度覆盖矩阵
  · Lin 审核每段，确认后推进下一段
  · 产出：完整的 24 段活动目录（~150 条条目）

阶段二：字段填充
  · 按确认的目录，逐段批量生成完整 v1.5 格式 JS 对象
  · 每条活动包含全部 12 个字段：
    id / domain / minAge / maxAge / dimFront / sensitivePeriod /
    whoFields / monFields / piaget / name / sub / theory /
    materials / environment / errorControl / steps / down / up /
    sci / observeAnchor / observeFocus / phases / linkedMilestones / theoryBite
  · 写入 activities-complete.js

阶段三：集成验证
  · 更新 app.js 引用（从旧文件切换到 activities-complete.js）
  · 浏览器 Golden Path 验收
  · 更新 CLAUDE.md 中的活动库现状说明
```

| 前缀 | 领域 | 起始月龄 |

## 3-6 岁内容策略

### 五大区域各段占比

| 月龄段 | 日常生活 P | 感官 S | 语言 L | 数学 N | 文化/科学 C | 每段合计 |
|--------|-----------|--------|--------|--------|------------|---------|
| 36-48 月 | 2-3 条 | 2 条 | 1-2 条 | 1 条 | 0-1 条 | 6-9 条 |
| 48-60 月 | 1-2 条 | 1-2 条 | 2 条 | 2 条 | 1 条 | 7-8 条 |
| 60-72 月 | 1 条 | 1 条 | 2 条 | 2-3 条 | 2 条 | 8-9 条 |

### 经典教具与家庭改编对照

| 区域 | 经典版（需购买/制作） | 家庭改编版 |
|------|---------------------|-----------|
| 感官 | 粉红塔（10 实木立方体） | 积木/书本叠高排序 |
| 感官 | 棕色梯 | 圆筒罐/纸卷排粗细 |
| 感官 | 色板一（三原色） | 彩纸方块配对 |
| 语言 | 砂纸字母板 | 手指在沙盘/盐盘描字母 |
| 数学 | 数棒 1-10 | 不同长度的筷子/积木条 |
| 数学 | 金色珠（十进制） | 豆子/石子分组 |

### 命名规范

- 经典教具版：名称后加 `(经典)`
- 家庭改编版：名称后加 `(家庭版)`
- 同一教具的进阶版：序号递增，名称附注难度
  - 示例：`S-36-01 粉红塔入门(经典)` → `S-48-01 粉红塔精进(经典)`

## 数据结构规范（v1.5 完整格式）

每条活动对象包含以下字段（与现有 activities-batch-0-6mo.js 保持一致）：

```js
{
  // A1. 基础信息
  id: 'P-36-01',
  domain: 'P',
  minAge: 36,       // 月龄
  maxAge: 48,
  dimFront: 'independence',
  sensitivePeriod: [...],
  whoFields: [...],
  monFields: [...],
  piaget: '...',

  // 展示信息
  name: '倒水练习',
  sub: '日常生活·36月起·...',
  theory: '...',

  // A2. 材料与环境
  materials: [...],
  environment: '...',
  errorControl: '...',

  // A3. 操作指引
  steps: [{ action: '...', note: '...' }],
  down: '...',
  up: '...',
  sci: '...',

  // A4. 观察引导（v1.5 新字段）
  observeAnchor: '...',           // ≤25字，第一人称视角
  observeFocus: [                 // 2-3条
    { look: '...', why: '...' }
  ],
  phases: [                       // 3阶段
    { range: [1, 2], label: '...', tip: '...' },
    { range: [3, 6], label: '...', tip: '...' },
    { range: [7, null], label: '...', tip: '...' }
  ],
  linkedMilestones: ['m_xxx_xxx'],
  theoryBite: '...'               // ≤30字+具体行为
}
```
