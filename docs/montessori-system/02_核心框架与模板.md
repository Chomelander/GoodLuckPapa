# 02 · 核心框架与模板

> 本文件是整套体系的**引擎文件**。所有活动的记录、评价、识别均复用此处定义的模板与规则。其他文件通过引用此处的编号（如"参见 02-B 记录表"）保持一致。

## A. 标准活动结构模板

每一个蒙氏活动必须严格按照以下结构撰写，缺项不得省略。

```markdown
## 活动名称：{{活动名}}

### A1. 基础信息
- 适配月龄/年龄：{{精准到月或半岁}}
- 核心敏感期：{{秩序 / 语言 / 感官 / 动作 / 社会性 / 细微事物}}
- 对应 WHO 发育领域（通俗描述）：{{粗大动作 / 精细动作 / 语言沟通 / 认知理解 / 社会情绪}}
- 蒙氏发展领域：{{日常生活 / 感官 / 语言 / 数学 / 科学文化}}

### A2. 有准备的环境与材料
- **材料清单**：列出每一项真实、可触摸、儿童尺寸、具备"错误控制"（能让孩子自己发现做错）的物品
- **环境布置**：固定摆放位置、儿童视角高度、独立取放动线、光线与噪声条件

### A3. 操作指引
- **成人示范要点**：极简、慢速、少语言（理想状态 0 语言，动作清晰可模仿）
- **儿童自主边界**：允许的探索方式 / 安全底线 / 何时介入（仅当涉及安全或伤害他人时介入）

### A4. 可观测行为指标
严格按 02-C 的 5 维度填写，每项须含**量化标准**（计时、计数、是/否判定）

### A5. 记录模板
直接套用 02-B 的标准记录表

### A6. 双维度评价标尺
- **WHO 维度**：参照 02-D，给出三级判定的量化依据
- **蒙氏维度**：参照 02-E，标注本活动主要发展的 1-3 个核心维度及三级水平判定依据

### A7. 兴趣点与优势识别
直接套用 02-F 与 02-G 的通用规则
```

## C. 可观测行为指标五维度

**核心原则**：所有指标必须能用眼睛看到、用秒表计时、用计数器累加，禁止"很开心""很感兴趣"这类主观词。

| 维度 | 量化标准 |
|---|---|
| **1. 自主选择** | 在环境中无成人提示下，主动走向并取出材料的**次数**（单日计） |
| **2. 专注时长** | 从拿起材料到完整放回原位的**连续操作秒数/分钟数**；中途被外部打断后是否在 30 秒内回归 |
| **3. 操作完成度** | 预设的 N 个核心步骤中完成的个数（如倒水共 5 步，完成 4 步） |
| **4. 拓展行为** | 是否出现非示范玩法（记录具体动作）；是否主动向他人展示或讲述发现（次数） |
| **5. 情绪稳定性** | 遇到挫折时：立即重试次数 / 暂停后恢复时长 / 求助方式 |

## D. 评价维度 1：WHO 发育进展

**WHO 五大发育领域**（通俗化命名）：
1. 粗大动作 —— 身体大肌肉运动
2. 精细动作 —— 手指手腕小肌肉运动
3. 语言沟通 —— 接收与表达
4. 认知理解 —— 解决问题、记忆、概念
5. 社会情绪 —— 与人互动、情绪调节、自我意识

**三级判定**（每级必须给出**量化依据**）：

| 等级 | 判定依据 |
|---|---|
| 符合预期 | 达到该月龄段该领域的典型表现（具体行为描述） |
| 超前发展 | 稳定达到下一月龄段的典型表现（连续 2 周以上可观察） |
| 需持续关注 | 未达到该月龄段下限的典型表现，且持续 4 周以上无进展 |

**重要**："需持续关注"**不等于**发育落后，它只是提示成人需要：① 检查环境准备是否适配；② 增加观察频次；③ 必要时咨询儿保医生。**禁止给儿童贴标签**。

## E. 评价维度 2：蒙氏十大发展维度

每个活动至少对应 1-3 个维度，每维度三级水平判定：

| 维度 | 萌芽 | 发展中 | 稳定表现 |
|---|---|---|---|
| **独立能力** | 需成人协助完成 | 能完成部分步骤 | 全程自主完成 |
| **秩序感** | 随意取放 | 多数时候归位 | 主动维护环境秩序 |
| **专注力** | 单次 < 典型时长 50% | 接近典型时长 | 超出典型时长，抗干扰 |
| **手眼协调** | 动作粗糙、频繁撒漏 | 偶有失误 | 动作精准 |
| **感官敏锐度** | 不区分差异 | 能区分明显差异 | 能区分细微差异 |
| **逻辑思维** | 无序尝试 | 有部分顺序 | 按步骤有逻辑地操作 |
| **语言表达** | 不表达 | 单词/短句表达 | 完整句描述操作 |
| **社会情绪** | 独占 / 冲突 | 能观察他人 | 主动协作或分享 |
| **自我纠错** | 不察觉错误 | 察觉但需提示 | 自主发现并调整 |
| **内在动机** | 需外部引导 | 偶尔自主选择 | 持续自主重复 |

### E.1 专注力质量 1–5 级刻度

> 引自 Lillard × Csikszentmihalyi 心流理论。此刻度专门用于月度复盘时对蒙氏"专注力"维度做更精细的定位。

| 等级 | 名称 | 行为指标 |
|---|---|---|
| 1 | 游荡（Wandering） | 在物品间快速切换，每个 < 30 秒，无目的 |
| 2 | 间歇注意（Intermittent） | 选择一个活动但频繁中断，易被分散 |
| 3 | 建构中（Developing） | 能维持 2–3 分钟，被打断后有时能回到活动 |
| 4 | 深度专注（Deep Concentration） | 维持 5+ 分钟，外界干扰时不中断，完成后平静满足 |
| 5 | 正常化（Normalized） | 长时间自主重复（10+ 分钟），完成后自行归位，表现出内在平静——蒙台梭利所描述"正常化"的核心表现 |

### E.2 秩序感 1–5 级刻度

| 等级 | 名称 | 行为指标 |
|---|---|---|
| 1 | 无明显需求 | 对环境变化无反应，物品随意放置 |
| 2 | 萌芽 | 固定流程被打破时偶有反应（不安/哭泣） |
| 3 | 敏感期活跃 | 坚持固定流程，对物品位置变化明显不安，开始自主归位 |
| 4 | 内化中 | 主动按固定位置归位，对他人"无序"行为表示纠正 |
| 5 | 已内化 | 自然维持秩序，不再因变化焦虑，具备灵活性 |

### E.3 典型时长参考（同龄儿童单次专注中位数）
- 0-1 岁：1-3 分钟
- 1-2 岁：3-6 分钟
- 2-3 岁：5-10 分钟
- 3-4 岁：10-15 分钟
- 4-5 岁：15-25 分钟
- 5-6 岁：20-30 分钟
- 6-9 岁：25-45 分钟
- 9-12 岁：30-60 分钟

## F. 兴趣点识别规则（四维度模型）

> 理论依据：Deci & Ryan 自我决定理论（SDT）+ Montessori "追随儿童"原则。兴趣点 = 内在需求的外在信号，通过四个维度交叉采集，避免单一指标误判。

### F.1 快速勾选版（日常用）

满足以下 **任意 2 项**，初步判定为当前兴趣点：

- ☐ 单日主动重复 ≥ 3 次
- ☐ 单次专注超出同龄典型 50% 以上
- ☐ 连续 3 天及以上主动选择
- ☐ 主动拓展非预设玩法 / 主动分享
- ☐ 遇挑战不放弃，尝试 2 种以上方式

### F.2 四维度交叉验证版（月度复盘用）

#### 维度一 · 自主选择频率

| 指标 | 信号强度 |
|---|---|
| 某类活动周频率 ≥ 4 次 | 强信号 |
| 自由游戏中反复出现的行为（≥ 3 次独立观察） | 强信号 |
| 对新活动主动接近 < 30 秒 | 中等信号 |

#### 维度二 · 投入深度

| 指标 | 信号强度 |
|---|---|
| 实际时长 > 建议时长 × 1.5 | 强信号 |
| 被打断后自主回到活动 | 强信号 |
| 同一动作自主重复 > 5 次 | 强信号 |

> M 视角补充：平静满足 ≠ 兴奋。当孩子反复进行某工作，脸上带着**平静**的满足，这是"正常化"信号，说明深度学习正在发生。

#### 维度三 · 情绪反应质量

| 指标 | 信号含义 |
|---|---|
| 活动中呈现**平静专注**（非亢奋） | 深层兴趣，优于兴奋状态 |
| 完成后满足表情 | 内在动机强 |
| 受阻时坚持 > 30 秒 | 目标承诺强 |

#### 维度四 · 自发迁移（最强信号）

| 指标 | 信号含义 |
|---|---|
| 将某领域行为迁移到其他场景 | 最强信号——概念已内化 |
| 自发创造新的活动变式 | 超越模仿，进入创造阶段 |

**示例**：孩子在容器游戏（S-6-03）反复放入/倒出 → 洗澡时用杯子舀水倒水 → 沙坑中铲沙装沙，即「容器-内含物」概念已迁移到 3 个场景，强信号确认。

### F.3 判定强度等级

| 等级 | 满足条件 |
|---|---|
| **强兴趣** | 4 维度中 ≥ 3 维度出现强信号 |
| **中等兴趣** | 2 维度出现强信号 |
| **观察中** | 仅 1 维度出现强信号 |

### F.4 判定后的行动

- 在环境中增加该方向的同类材料
- 延长可用时段
- **不干扰、不夸奖**，让兴趣自然生长（外部奖励会削弱内在动机——Lillard 引用 Deci & Ryan 实证结论）

## G. 优势领域识别规则

### G.1 判定条件（需同时满足）

1. 在同一类活动中表现持续稳定超出同龄典型水平
2. 持续观察窗口 ≥ 4 周（月度确认）/ ≥ 12 周（季度正式确认）
3. 在至少 3 个不同具体活动中均有此表现

### G.2 判定后的标注格式

```
优势领域：{{蒙氏维度}} + {{WHO 领域}}
延伸方向：{{推荐的下一阶段蒙氏教具线/活动线}}
```

### G.3 兴趣信号 → 优势领域映射表

> 理论依据：Gardner 多元智能理论 × 蒙氏领域 × 丽乐德实证。识别信号模式后，对照下表确定延伸方向，**不是为了给孩子定性，而是为了准备更丰富的环境**。

| 反复出现的兴趣信号模式 | Gardner 智能 | 蒙氏领域 | 下一步环境准备方向 |
|---|---|---|---|
| 反复操作容器、排列、分类、嵌套 | 逻辑数学 + 空间 | 感官教育 | 嵌套/排序教具 → 形状配对盒 → 数学教具（3 岁+） |
| 对节奏、音调、旋律敏感；自发拍打节奏 | 音乐 | 感官-听觉 + 文化-音乐 | 简单打击乐器 → 节奏模仿游戏 → 音乐欣赏时段 |
| 大运动持续高活跃；探索空间边界 | 身体动觉 | 运动 | 攀爬/平衡设施 → 拓展户外场地 → 工具使用（扫帚、水桶） |
| 语言输出超前；爱听故事；主动命名 | 语言 | 语言 | 丰富真实物品词汇卡 → 绘本（真实照片类）→ 对话扩展 |
| 对他人情绪高度敏感；主动安慰 | 人际 | 日常生活-社交礼仪 | 角色扮演材料 → 混龄社交机会 → 礼仪活动 |
| 长时间自我专注某任务；抗拒打断 | 内省 | 日常生活-专注工作 | 尊重独处时间 → 扩展工作毯范围 → 减少外部打断 |
| 对自然事物（虫、叶、花、土）着迷 | 自然观察 | 文化-生物/自然 | 户外探索时段 → 动植物真实标本/卡片 → 简单栽培 |

### G.4 底线

- 优势识别的目的是**为下一阶段的环境做准备**，不是让孩子"成为什么样的人"
- 不告诉孩子他"有天赋"
- 不向他人展示优势报告
- **候选优势**（4 周）→ **确认优势**（12 周连续）→ 更新环境。确认后仍只做环境准备，不设目标

## H. 活动编号体系

> 所有活动卡统一采用 `{领域}-{月龄}-{序号}` 格式，便于跨文件引用、追踪与检索。

| 前缀 | 领域 | 英文 |
|---|---|---|
| P | 日常生活 | Practical Life |
| S | 感官教育 | Sensorial |
| M | 运动 | Movement |
| L | 语言 | Language |
| E | 社会情绪 | Emotional / Social |
| C | 文化（3 岁+） | Cultural Extensions |
| N | 数学（3 岁+） | Numeracy / Mathematics |

**月龄位**：以活动可引入的下限月龄数字表示。
- `S-6-02` = Sensorial 类、6 月起可用的第 2 号活动 = 宝藏篮
- `P-12-03` = Practical Life 类、12 月起可用的第 3 号活动 = 倒水练习

**引用示例**：在记录表的"活动名称"栏可直接写 `S-6-02 宝藏篮`。
