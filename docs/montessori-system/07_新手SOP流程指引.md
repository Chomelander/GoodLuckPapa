## <a id="part-1"></a>1. 心态三件套

在翻开任何文件之前，把这 3 句话抄在冰箱上或手机锁屏：

> 1. 我是**观察者**，不是老师
> 2. 我调整**环境**，不调整孩子
> 3. 数据服务于决策，**不服务于我的情绪**

**新手三大陷阱**：

| 陷阱 | 症状 | 破解 |
|---|---|---|
| 把观察当成打分 | 焦虑、反复翻记录本 | 只和自己 2 周前对比 |
| 把记录本当朋友圈 | 发群、炫耀、比较 | 记录本只给儿保医生看 |
| 把"符合预期"当目标 | 鸡娃、催促 | 读 `01` 第六节"何时停下记录" |

任一出现 → **立即合上记录本 1 周，只纯粹陪伴**。

## <a id="part-2"></a>2. 完整闭环总览图

```
┌─────────────────────────────────────────────────────────────────┐
│                    首次使用（一次性·阶段 A）                       │
│  认知准备 → 儿童定位 → 活动选择 → 环境准备 → 心理就绪             │
└───────────────────────────┬─────────────────────────────────────┘
                            │
                            ↓
┌─────────────────────────────────────────────────────────────────┐
│              每次活动的固定 7 步 SOP（阶段 B · 日常循环）          │
│                                                                 │
│   ①就位 → ②等待 → ③示范 → ④退后 → ⑤观察 → ⑥收尾 → ⑦记录         │
│                                                                 │
│                        （每日 1-3 次）                           │
└───────────────────────────┬─────────────────────────────────────┘
                            │
                            ↓
┌─────────────────────────────────────────────────────────────────┐
│                      复盘与调整（阶段 C）                         │
│                                                                 │
│   周复盘（15 分钟） → 月复盘（30 分钟） → 季复盘（60 分钟）        │
│                                                                 │
│                      输出：环境调整清单                           │
└───────────────────────────┬─────────────────────────────────────┘
                            │
                            ↓
                   回到阶段 B · 下一轮循环
```

三个关键原则：

- **阶段 A 只走一次**，之后不再重复
- **阶段 B 是心跳**，只要你在观察孩子，就在这个循环里
- **阶段 C 服务于阶段 B**，复盘的唯一目的是优化下一轮的环境

## <a id="part-3"></a>3. 阶段 A · 首次使用启动流程

一次性完成，约 2 小时，建议分 2-3 天。

### A-1 · 认知准备（30 分钟）

| 顺序  | 做什么                | 文件               | 时长   |
| --- | ------------------ | ---------------- | ---- |
| 1   | 读"核心初衷" + "使用原则"   | `00_体系说明.md`     | 10 分 |
| 2   | 读"成人三重角色" + "六条底线" | `01_通用观察评估准则.md` | 10 分 |
| 3   | 精读 B 记录表 + C 五维度指标 | `02_核心框架与模板.md`  | 10 分 |

**产出**：脑中形成三件套「角色定位 + 禁区底线 + 记录模板」

**检查点**：能脱口说出"我不说真棒""我不替他做""我只记事实"

---

### A-2 · 儿童定位（5 分钟）

打开 `03_年龄段索引.md`，找到孩子当前月龄对应的 **3 月小节**。

**抄到手机备忘录**：

```
我家宝宝当前阶段：___-___ 月
核心敏感期：________________
本阶段主导蒙氏维度：____________
推荐活动方向（3 条）：
  1. _____________
  2. _____________
  3. _____________
```

这张卡是你的"北极星"，未来 3 个月你只做这 3 个方向的活动。

---

### A-3 · 首发活动选择（10 分钟）

**选择原则**（小白第一次，选最简单的）：

```
从 3 个推荐方向中，挑 ❶ 材料最容易获取 ❷ 操作最简单 的 1 个活动
```

打开 `04_示例活动集_0-1岁.md`（或对应年龄段文件），找到该活动的完整说明。

**只读 4 个部分**：
- A2 材料清单 → 写购物单
- A3 操作指引 → 理解示范步骤
- A4 观察重点 → 知道明天看什么
- A5 记录要点 → 复制到备忘录

---

### A-4 · 环境与材料准备（30-60 分钟 · 宝宝不在场）

**四步准备法**：

```
Step 1 · 采购/自制材料
  ↓
Step 2 · 安全检查（尺寸、边角、无毒）
  ↓
Step 3 · 固定位置（选一个永远不变的角落）
  ↓
Step 4 · 清场（移走该区域所有会发声/闪光的玩具）
```

**材料安全红线**：

| 项目 | 红线 |
|---|---|
| 小部件尺寸 | 直径 ≥ 4cm（塞不进卫生纸卷筒） |
| 材质 | 实木 > 不锈钢 > 玻璃 > 陶瓷 >>> 塑料 |
| 涂层 | 无漆或食品级蜡 |
| 边角 | 无锐利，圆润处理 |

---

### A-5 · 心理就绪检查（5 分钟）

在备忘录写下**明天要做的 3 件事** + **3 件不做的事**：

```
✓ 要做：
  1. 慢动作示范 1 次（60 秒内）
  2. 示范后退后 1 米
  3. 用秒表记专注时长

✗ 不做：
  1. 不说"真棒""乖""对了"
  2. 他失败时不伸手帮
  3. 他重复 20 次也不换新玩具
```

**就绪**。明天开始进入阶段 B。

## <a id="part-4"></a>4. 阶段 B · 每次活动的固定 7 步 SOP

**这是整个体系的心跳**。每次活动都按这 7 步走，直到成为肌肉记忆。

### 快速流程图

```
┌───────┐   ┌───────┐   ┌───────┐   ┌───────┐
│ ①就位 │ → │ ②等待 │ → │ ③示范 │ → │ ④退后 │
└───────┘   └───────┘   └───────┘   └───────┘
                                         │
                                         ↓
┌───────┐   ┌───────┐   ┌───────┐
│ ⑦记录 │ ← │ ⑥收尾 │ ← │ ⑤观察 │
└───────┘   └───────┘   └───────┘
```

---

### 第 1 步 · 就位（1 分钟）

**动作**：
- 检查材料是否在固定位置
- 检查环境（光线、无干扰源、手机静音收远）
- 自己坐到观察位（孩子侧前方 1-1.5 米）

**不做**：喊孩子"宝宝来玩这个"

---

### 第 2 步 · 等待（不限时）

**核心**：**让活动发起权回到孩子手里**。

- 如果是**新活动第一次**：跳到第 3 步示范
- 如果是**已示范过的活动**：等孩子自己走/爬过来

**关键心法**：即使孩子半小时都不来，这也是**有效观察**——你记录"今日自主选择次数 = 0"。这个数据本身就是信息。

---

### 第 3 步 · 示范 or 不示范

**判断规则**：

```
IF 这是新活动且孩子还没见过
THEN 完整慢动作示范 1 次（严格按活动文件 A3）
     · 全程 0-1 句话
     · 每个动作慢到"不自然"
     · 示范完立即停手

IF 这是旧活动
THEN 不示范，直接跳到第 4 步
```

**示范禁忌**：

| ❌ 不要 | ✓ 改成 |
|---|---|
| 边做边讲"看，这样放进去~" | 闭嘴，只做动作 |
| 做完问"你也来试试？" | 把材料放回原位，退后 |
| 孩子没看懂再示范第二次 | 一次就够，允许他自己摸索 |
| 示范时孩子抢 | 把材料交给他，结束示范 |

---

### 第 4 步 · 退后（5 秒内）

**动作**：
- 起身或挪到孩子侧后方 1-1.5 米
- 保持坐姿（不居高临下）
- 视线不直视孩子（可看向侧方）
- 拿出秒表 / 记录本

**退后的心理意义**：用身体语言告诉孩子「现在是你的时间」。

---

### 第 5 步 · 观察计数（5-30 分钟）

**只盯 5 个指标**（参见 02-C）：

| 指标 | 怎么看 |
|---|---|
| 自主选择 | 他是否主动取材料？ |
| 专注时长 | 秒表计时：从取到放回的连续秒数 |
| 操作完成度 | 完成核心步骤的个数 |
| 拓展行为 | 有无预设之外的动作？ |
| 情绪稳定性 | 遇到挫折时的反应 |

**5 大纪律**：

```
🚫 不说话（除非求助或安全）
🚫 不伸手
🚫 不叹气
🚫 不摇头
🚫 不拍照（会让孩子意识到被观察）
```

**例外**：孩子真的寻求帮助时，给**最小必要支持**，然后立即退回观察位。

---

### 第 6 步 · 平静收尾（2 分钟）

**触发收尾的信号**：
- 孩子主动离开活动区超过 1 分钟
- 孩子明显困倦、烦躁
- 活动时间已达 30 分钟

**收尾动作**：

```
1. 平静地说"我们把它放回去吧"（或干脆不说，自己示范归位）
2. 把材料归回固定位置
3. 不评价刚才的表现
4. 不说"下次我们再玩"之类的暗示
```

**禁忌**：
- ❌ "哎呀你怎么不收拾"
- ❌ "你真棒玩了这么久"
- ❌ "明天我们一定要完成哦"

---

### 第 7 步 · 记录（≤ 5 分钟）

**在宝宝不在场时**（午睡时 / 晚上），打开备忘录填 02-B 记录表：

```
日期：____-__-__
月龄：___月
活动：____________
发起方式：☐自主 ☐邀请 ☐主导
专注时长：___ 分 ___ 秒
累计次数：___
完成度：☐完整 ☐部分 ☐未完成
自主纠错：___ 次
重复次数：___ 次
困难：__________
应对：☐自主 ☐求助 ☐放弃
拓展表现：__________
情绪：☐平稳 ☐波动 ☐需支持
```

**填写 3 大禁区**：

| ❌ 禁写 | ✓ 改成 |
|---|---|
| "他很开心" | "嘴角上扬 ≥ 3 次" |
| "他不喜欢" | "主动选择次数 = 0" |
| "进步了" | "比上周专注时长增加 2 分钟" |

## <a id="part-5"></a>5. 阶段 C · 复盘 SOP

### 周复盘（每周日晚，15 分钟）

**5 问自检**：

```
1. 本周总活动次数？          ___ 次
2. 自主选择占比？             ___%
3. 最长专注时长？             ___ 分钟
4. 出现过哪些拓展行为？        ___
5. 对照 02-F，触达兴趣点规则的项数？ ___ / 5
```

**行动规则**：

| 触达项数 | 下周怎么办 |
|---|---|
| 0-1 | 保持环境不变，继续观察 |
| 2-3 | 在相同方向增加 1 件新材料 |
| 4-5 | 确认为**当前兴趣点**，记录到手机备忘录 |

---

### 月复盘（每月末，30 分钟）

**开打 `06_阶段汇总评估框架.md` 第一节**，依次填写：

```
Step 1 · 月度表头（月龄、有效记录天数）
    ↓
Step 2 · 活动参与总表（所有活动汇总）
    ↓
Step 3 · 兴趣点候选清单
    ↓
Step 4 · 蒙氏十维度雷达（1-3 分打分）
    ↓
Step 5 · WHO 五领域对照
    ↓
Step 6 · 下月环境调整清单 ⭐
    ↓
Step 7 · 成人自我调整自检（6 问）
```

**输出物**：下个月的**环境调整清单**（这是月复盘的核心产出，没有这个等于白做）。

---

### 季复盘（每季度末，60 分钟）

**做三件事**：

1. **雷达对比**：把 3 个月的蒙氏十维度雷达叠加，看**形状变化**
2. **兴趣演变追踪**：哪些兴趣点深化、哪些衰减、哪些转向
3. **季度叙事**：用 200 字写一段**事实性描述**（不是评价）

**季度叙事模板**：

```
本季度（___月龄 → ___月龄）：
· 核心观察：_________________
· 稳定出现的能力：_________________
· 新兴的兴趣方向：_________________
· 尚未出现的典型行为：_________________
· 下季度环境调整：_________________
```

## <a id="part-6"></a>6. 决策树 · 遇到问题怎么办

### 场景 1 · 宝宝完全不理新活动

```
已示范且等待 ≥ 3 天？
  ├─ 是 → 继续等待 4 天后再决策
  │         ├─ 仍无反应 → 把该活动收起（不是永远，2 周后再放回）
  │         └─ 有反应 → 进入正常观察流程
  └─ 否 → 继续等待，不主动引导
```

**绝对不做**：把活动端到宝宝面前、发出惊呼吸引注意。

---

### 场景 2 · 宝宝重复 50 次我快疯了

**这不是问题，是蒙氏的核心价值**。
但如果你真的受不了 → 戴耳机、做自己的事，让他继续。

**重复 = 孩子在把这件事内化为自己的一部分**。

---

### 场景 3 · 宝宝把材料"用错"了

```
涉及安全或伤害他人？
  ├─ 是 → 平静收起活动，说"现在不是 XX 时间"
  └─ 否 → 什么都不做，记录到"拓展行为"栏
```

**案例**：
- 把水倒到桌上 → 拓展行为，让他自己擦
- 把盒子当鼓敲 → 拓展行为，探索期合法
- 把小球扔向奶奶 → 涉及他人，收起

---

### 场景 4 · 数据难看，我开始焦虑

**三步自救**：

```
1. 合上记录本 1 周
2. 重读 01 第六节「何时停下记录」
3. 这一周只做一件事：陪伴，不带目的
```

然后回来，你会发现数据本身并没有变，只是你看数据的心态变了。

---

### 场景 5 · 其他家人（老人/配偶）不认同

**优先级**：
1. 保护孩子的**专注时刻**不被打断（最核心底线）
2. 保护**固定的环境位置**不被移动
3. 其他细节可以妥协

**沟通原则**：
- 不讲理论（蒙氏、丽乐德对方听不懂）
- 只讲观察到的**具体行为**（"你看他刚才 10 分钟都没离开"）
- 不要求对方改变，只请他们"不打扰"

---

### 场景 6 · 孩子哭了要不要抱

```
哭声性质？
  ├─ 生理需求（饿、困、痛） → 立即回应
  ├─ 情感需求（害怕、求抱） → 立即回应
  └─ 挫败感（塞不进、拿不到） → 先观察 5-10 秒
        ├─ 自己平复 → 继续观察
        └─ 寻求帮助 → 给最小支持后退回
```

**蒙氏不等于冷漠**。情感需求永远优先于"让他自己解决"。

## <a id="part-7"></a>7. 新手首月打卡清单

### Week 1 · 启动

- [ ] Day 1：完成阶段 A 的 A-1 认知准备
- [ ] Day 2：完成 A-2 儿童定位 + A-3 活动选择
- [ ] Day 3：完成 A-4 环境准备（采购、布置）
- [ ] Day 4：首次示范 + 首次观察
- [ ] Day 5-7：连续 3 天日常观察 + 记录
- [ ] Day 7 晚：第一次周复盘（5 问自检）

### Week 2 · 稳固

- [ ] 每天 1 次观察 + 记录
- [ ] 识别首个兴趣点候选（如有）
- [ ] Day 14 晚：周复盘

### Week 3 · 扩展

- [ ] 如出现兴趣点 → 增加 1 件同方向材料
- [ ] 引入第 2 个活动（不撤掉第 1 个）
- [ ] Day 21 晚：周复盘

### Week 4 · 复盘

- [ ] 继续日常观察
- [ ] Day 28 晚：第一次**月复盘**（使用 06 文件）
- [ ] 输出下月环境调整清单

### 首月完成后的自我奖励

- 给自己（不是孩子）买一杯咖啡
- 在记录本扉页写一句话："我坚持了一个月"
- 把月度复盘的数据归档（纸质装订或云盘分文件夹）

## 终极心法

这套 SOP 最终只为一件事服务：

> **让你的眼睛学会看见孩子的真实节律，而不是看见你对孩子的期待。**

流程是拐杖，不是终点。当有一天你发现：
- 你不再需要记录本也能准确判断孩子的状态
- 你不再焦虑数据好坏
- 你的第一反应是"调整环境"而不是"纠正孩子"

那时候你就可以把这套 SOP 合上了——因为**它已经成为你的一部分**。
