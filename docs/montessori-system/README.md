# 蒙台梭利家庭观察体系 · 使用指南

> 一套面向家长的蒙台梭利教育活动、观察与评估体系。
> 基于玛丽亚·蒙台梭利 + 安吉琳·S. 丽乐德双视角，配套 WHO 发育里程碑。

---

## 这是什么

不是育儿任务清单，是一份**观察手册**。帮你把目光从"该教孩子什么"转向"孩子此刻正在向我展示什么"。

**核心输出**：
- 可直接复用的标准化活动模板与编号体系
- 可观测、可记录、可评价的日常观察工具
- WHO 发育 + 蒙氏十维度的双轨评估
- 兴趣点（4 维度模型）与优势领域识别规则
- 月度 / 季度 / 年度复盘框架
- **新手父母的 7 步日常 SOP**

## 活动编号体系（02-H）

所有活动统一采用 `{领域}-{月龄}-{序号}` 格式：

| 前缀 | 领域 | 英文 |
|---|---|---|
| P | 日常生活 | Practical Life |
| S | 感官教育 | Sensorial |
| M | 运动 | Movement |
| L | 语言 | Language |
| E | 社会情绪 | Emotional / Social |
| C | 文化（3 岁+） | Cultural |
| N | 数学（3 岁+） | Numeracy |

**示例**：
- `S-0-01` = 感官类 / 0 月龄起可用 / 第 1 号 = 视觉动态挂件
- `S-6-02` = 感官类 / 6 月龄起可用 / 第 2 号 = 宝藏篮
- `S-6-04` = 感官类 / 6 月龄起可用 / 第 4 号 = 物体永久性盒 ⭐
- `M-9-04` = 运动类 / 9 月龄起可用 / 第 4 号 = 推拉学步与搬运任务

记录表的"活动名称"栏可直接写编号，便于跨文件检索。

## 兴趣点识别（四维度交叉验证）

**快速勾选版**（日常用）——满足 **任意 2 项** 即初判为兴趣点：

```
  ☐ 单日主动重复 ≥ 3 次
  ☐ 单次专注超出同龄典型 50% 以上
  ☐ 连续 3 天及以上主动选择
  ☐ 主动拓展非预设玩法 / 主动分享
  ☐ 遇挑战不放弃，尝试 2 种以上方式
```

**月度复盘版**（四维度交叉验证）：自主选择频率 / 投入深度 / 情绪反应质量 / 自发迁移（最强信号）→ **详见 02-F.2**

判定后做什么：**增加同方向材料**、**延长可用时段**、**不夸奖、不干扰**。

## 六条底线（必须守住）

```
┌───────────────────────────────────────────┐
│  1. 不奖励（包括"真棒""聪明"）             │
│  2. 不惩罚                                 │
│  3. 不比较（不与他人，不与过去的期望）       │
│  4. 不贴标签（不说"慢热""有天赋"等）        │
│  5. 不打断（孩子专注中成人闭嘴）             │
│  6. 不展示（记录本不上朋友圈）               │
└───────────────────────────────────────────┘
```

任何一条被突破，记录数据就开始反向伤害孩子。

## 两个 04 文件的分工

很多人会问：为什么有两个 04？它们如何配合？

| 文件 | 角色 | 何时用 |
|---|---|---|
| `04_示例活动集_0-1岁.md` | **框架索引**：按 4 个 3 月子阶段组织，含维度覆盖矩阵与整体地图 | 选活动前先看这里，定位方向 |
| `04_示例活动_0-1岁.md` | **详细卡库**：25 张扩展活动卡 + 理论引用（Piaget、Lillard、Meltzoff、Tomasello） | 选定方向后翻这里查操作细节 |

**使用顺序**：先看**集**的矩阵定位方向 → 再翻**卡库**查细节 → 按 07 的 7 步心跳执行。

## 三个容易犯的错

| 错法 | 正解 |
|---|---|
| 把记录表填得密密麻麻、情绪化描述 | 只记可观察事实，11 个字段就够 |
| 追求蒙氏十维度雷达"画出正圆" | 每个孩子有自己的形状，只和自己比 |
| 看孩子没按"预期"就想换活动 | 重复 50 次是正常的工作，让他做 |

## 最后

这套体系最终只为一句话服务：

> **让你看见你的孩子，而不是看见你对孩子的期待。**

如果某一天你发现数据都漂亮但孩子眼中的光淡了——**立刻合上记录本，去陪他**。
