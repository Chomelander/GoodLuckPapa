## 二、蒙氏十维度月度雷达

将本月所有活动的评价结果汇总，在十个维度上标出"整体水平"。

**刻度说明**：
- **8 个维度**（独立/手眼协调/感官/逻辑/语言/社会情绪/自我纠错/内在动机）使用三级刻度：萌芽（1）/ 发展中（2）/ 稳定（3）
- **专注力**使用 `02-E.1` 的**专注力 5 级刻度**（游荡→间歇→建构中→深度专注→正常化），更细致地捕捉蒙氏"正常化"过程
- **秩序感**使用 `02-E.2` 的**秩序感 5 级刻度**（无→萌芽→敏感期活跃→内化中→已内化），识别秩序敏感期的具体位置

```
                独立能力
                  │
   内在动机 ─────┼───── 秩序感
              ╱ │ ╲
   自我纠错 ╱   │   ╲ 专注力
          │     │     │
   社会情绪     │    手眼协调
          │     │     │
   语言表达 ╲   │   ╱ 感官敏锐度
              ╲ │ ╱
            逻辑思维
```

**填写方法**：每个维度三级水平（萌芽 / 发展中 / 稳定），用 1-3 打分，连线成一个形状。

**核心警示**：
- **不追求"正圆"**。均衡发展是平均值，每个孩子都有自己的节奏与形状
- **不与他人的雷达比较**，只与自己上个月的雷达比较
- **不是越外圈越好**，"萌芽"维度也是值得被看见的成长

## 四、个性化建议生成逻辑

根据月度汇总结果，**成人**给**自己**（不是给孩子）的下月行动清单：

### 4.1 环境调整建议

**基于兴趣点**：
```
IF 某活动被判定为兴趣点
THEN 在环境中增加 1-2 件该方向的同类/进阶材料
AND 保证该活动每日可用时段 ≥ 30 分钟
AND 不夸奖、不催促、不干扰重复
```

**基于优势领域**：
```
IF 某类活动被判定为优势领域
THEN 查阅 03_年龄段索引.md 的下一阶段推荐活动
AND 引入 1 件下一阶段的起始教具作为"试探"
AND 观察 2 周，若孩子主动选择则继续，若无反应则撤回
```

**基于需持续关注项**：
```
IF 某项连续 3 周无进展
THEN 步骤 1：检查环境准备（高度、秩序、干扰源）
     步骤 2：减少材料总数（宁少勿多）
     步骤 3：增加"无活动"自由时段
     步骤 4：若连续 6 周仍无变化，咨询儿保医生
ABSOLUTELY NOT：增加训练、更多示范、口头鼓励、奖励机制
```

### 4.2 成人自我调整建议

这一部分是**给成人的镜子**：

| 自检问题 | 本月我做得如何 |
|---|---|
| 我是否在每次观察时做到了"少语言"？ | ☐ 是 ☐ 偶有提醒 ☐ 需改进 |
| 我是否在孩子"做错"时没有立即介入？ | |
| 我是否没有在他人面前讨论孩子的记录？ | |
| 我是否没有使用"真棒""聪明"这类词？ | |
| 我是否把孩子和其他孩子比较了？ | |
| 我是否因为数据而焦虑？ | |

**如果出现 3 项以上"需改进"**：建议暂停记录 1 周，只纯粹陪伴，不带目的。

## 五、季度汇总（每 3 个月一次）

在月度汇总基础上，季度末额外完成：

### 5.1 连续 3 个月雷达对比

将 3 个月的蒙氏十维度雷达**叠加**在一张图上（不同颜色），观察**形状的变化**而非分数的高低。

### 5.2 兴趣点演变追踪

| 兴趣点 | 首次出现月份 | 持续月数 | 当前状态 | 延伸方向 |
|---|---|---|---|---|
| | | | ☐ 深化中 ☐ 稳定 ☐ 衰减 ☐ 转向 | |

**重要**：兴趣点的"衰减"不是坏事，可能意味着该敏感期正在关闭，孩子要迈向下一阶段。

### 5.3 优势领域确认

连续 3 个月（即 12 周）持续稳定表现的领域，方可从"候选优势"正式确认为"优势领域"。确认后仍需遵守：

- 不告诉孩子他"有天赋"
- 不向他人炫耀
- 不设"目标"
- 只做一件事：**为下一阶段的环境做准备**

### 5.4 季度回顾叙事（一段话）

用 200 字以内写下本季度对孩子的**事实性观察叙事**：

> 示例：
> "本季度 42-45 月龄，独立完成倒水循环从 6/9 步提升到 9/9 步的出现从 3 月第 2 周开始稳定。日常生活类活动的自主选择次数比上季度增长 60%。出现了一个新的重复模式：对所有有盖容器的'拧开-关闭'动作有持续兴趣（4 月 12 日首次出现，至今连续 18 天）。本月没有出现明显的情绪调节退行。下季度环境调整方向：在日常生活区增加 3 件不同拧法的真实容器，减少托盘数量从 8 件到 6 件。"

**禁止**：
- "他变得更聪明了"
- "她比 XX 家的孩子进步快"
- "我觉得他在 XX 方面有天赋"

## 六、年度评估（可选，每 12 个月一次）

年度评估只做三件事：

1. **回顾 12 个季度叙事**，整理出**孩子的成长故事线**（不是简历）
2. **归档本年度的所有记录本**（装订，不销毁——孩子 18 岁后可作为礼物归还）
3. **写一封信给自己**：这一年作为观察者，我学到了什么？

> "观察儿童的成人，自己也在被观察——被自己内心那个曾经也是孩子的自己。"

```
║  □ 是  □ 需要我更多陪伴，少一点记录                           ║
╚══════════════════════════════════════════════════════╝
```

## 八、使用本框架的核心心法

1. **数据服务于决策，不服务于情绪**
2. **每一个数字背后是一个正在成长的完整的人**
3. **复盘的目的是调整环境，不是调整孩子**
4. **最重要的数据是：孩子本月的笑容是否依然松弛、眼神是否依然明亮**

如果所有数据都漂亮，但孩子眼中的光消失了——**立即停下一切评估系统**，回到最根本的事：陪伴、信任、爱。
