# 01 · 通用观察与评估准则

> 本文件定义使用本体系时成人必须遵守的角色定位与行为底线。所有具体活动的开展都建立在这些准则之上。

---

## 一、成人的三重角色

蒙台梭利明确指出：在儿童的工作场景中，成人只能扮演三种角色，**绝不包括"主导者"或"教导者"**。

### 角色 1：观察者（Observer）

- **核心动作**：退后、安静、记录事实
- **不做什么**：不点评、不暗示、不打断、不替孩子完成
- **具体表现**：
  - 与孩子保持至少 1 米距离
  - 手持记录本，只记录可观察的行为（参见 02-C 指标）
  - 面部表情保持中性（不皱眉、不惊呼、不大笑）

### 角色 2：环境准备者（Preparer of the Environment）

- **核心动作**：在孩子不在场时，精心准备材料与空间
- **检查清单**：
  - [ ] 材料是否真实（真陶瓷而非塑料玩具，真水而非模拟）
  - [ ] 材料尺寸是否适合儿童的手
  - [ ] 材料是否具备自我纠错属性
  - [ ] 摆放位置是否固定且儿童可独立取放
  - [ ] 动线是否清晰无障碍
  - [ ] 同一区域是否只呈现一个新挑战

### 角色 3：极简引导者（Minimal Guide）

- **核心动作**：仅在孩子**主动请求**或**安全风险出现**时介入
- **介入原则**：
  - 语言极简（一句话以内）
  - 动作缓慢（是示范不是代劳）
  - 示范完立即退回观察者位置
- **禁止**：未被请求就介入、边做边讲解、重复示范同一步骤超过 2 次

## 二、观察的注意事项

### 观察窗口

| 观察类型 | 推荐窗口 | 频次 |
|---|---|---|
| 单次活动观察 | 完整一次操作周期（从取出到归位） | 每次活动 |
| 日观察 | 一天内所有主动选择的活动 | 每日 1 次汇总 |
| 周观察 | 一周内兴趣点趋势 | 每周 1 次汇总 |
| 月观察 | 配合 06 文件的月度评估 | 每月 1 次 |

### 观察的常见误区

| 误区 | 修正 |
|---|---|
| 观察时忍不住提问"你在做什么？" | 闭嘴。问题 = 打断 |
| 看到孩子"做错"就想纠正 | 只要不涉及安全，让材料自己教 |
| 孩子哭就立刻冲过去 | 先观察 5-10 秒，确认是挫败还是真的需要 |
| 把观察结果和其他孩子比较 | 只和自己 2 周前的记录比较 |
| 用"聪明"、"笨"这类词记录 | 只记录行为，不记录评价 |

### 观察时的身体姿态

- 坐在低凳或地板上，视线**不高于孩子**
- 避免正面直视，侧身或斜后方更不具压迫感
- 手机放远处（来电会打断孩子的专注时刻）

## 三、评价的核心原则

### 评价是为了环境调整，不是为了评判儿童

每一次评价后，必须回答的唯一问题是：

> **"我接下来需要为他/她调整环境的哪一部分？"**

如果你的评价后没有环境调整动作，这个评价就是无效的（或者，它变成了伤害）。

### 评价的三条底线

1. **禁止给儿童贴标签**
   - ✗ "他是个慢热的孩子"
   - ✗ "她在数学方面有天赋"
   - ✓ "在过去 4 周的 12 次数量配对活动中，稳定出现超出典型时长 60% 以上的专注"

2. **禁止跨儿童比较**
   - ✗ "比隔壁家 XX 差/好"
   - ✓ "与他自己 3 月份的记录相比，独立取放行为从 20% 上升到 70%"

3. **禁止奖惩性语言**
   - ✗ "你做得真棒！"（即使是发自内心的赞美）
   - ✓ "你把每一颗豆子都放进了瓶子里。"（描述事实）
   - 理由：赞美把控制权从孩子转移到成人，削弱内在动机

## 四、记录的隐私与尊重

### 记录本的使用规则

- 记录本是**工具**，不是**展示品**
- 不拍照发朋友圈、亲戚群、家长群
- 不在孩子能听到的范围内讨论记录内容
- 孩子 6 岁后，如果他想看自己的记录本，**完全告知**，并允许他提出修改或删除

### 与他人分享时的原则

| 对象 | 可以分享 | 不应分享 |
|---|---|---|
| 配偶/共同抚养人 | 全部 | — |
| 儿保医生 | 全部（辅助临床判断） | — |
| 蒙氏教师 | 全部（协作观察） | — |
| 其他家长 | 体系与方法论 | 具体儿童的数据 |
| 亲戚朋友 | 无 | 一切 |

---

## 五、使用本体系的 5 条自我提醒

在每次打开记录本之前，默念一遍：

1. 我今天来是**观察**的，不是来**教**的
2. 我的任务是调整**环境**，不是调整**孩子**
3. 孩子的重复不是卡住，是在**工作**
4. 我不用"聪明"评价他，我用**事实**记录他
5. 这个记录本永远不会离开我，**他是一个完整的人**

---

## 六、何时停下记录？

如果出现以下任一情况，请**暂停使用本体系 1-2 周**：

- 你开始因为记录本的数据焦虑
- 你发现自己在拿孩子和任何人比较
- 你开始"安排"活动让孩子"达标"
- 孩子表现出对记录本或观察的抗拒

体系是服务成人观察能力的，不是反过来奴役成人和儿童。**当它变成压力源，它就失效了**。
