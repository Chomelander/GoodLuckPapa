# MVP 功能清单

## 产品 MVP 范围

本产品 MVP（最小可行产品）包含 **5 个核心功能**，全部已实装。这些功能直接对应 docs/01 中的三个核心痛点解决方案。

---

## Must Have 清单（核心功能）

### 1. 成长档案 — 解决"观察无体系"痛点
**功能描述**：
- 为每个孩子建立专属观察档案
- 记录观察笔记、照片、发展里程碑
- 按蒙台梭利 8 大模块组织观察内容
- 支持离线记录，自动同步

**实装状态**：✅ 已完成
- 前端: moments.js（时光页面）
- 后端: REST API for child profiles
- 测试: 44/244 测试通过（与档案相关）

**核心价值**：
- 帮助家长建立结构化的观察习惯
- 提供框架化的观察维度指导

---

### 2. 活动推荐 — 解决"活动库不适龄"痛点
**功能描述**：
- 根据孩子月龄推荐蒙台梭利适龄活动
- 143 个原创活动卡片（0-72 月龄）
- 按蒙台梭利 8 大模块分类：日常生活、感官、数学、语言、文化、艺术、音乐、运动
- 可标记已完成活动和孩子反馈

**实装状态**：✅ 已完成
- 前端: record.js（活动记录）+ 内嵌活动库
- 后端: activities-complete.js（143 个活动定义）
- 测试: 68/244 测试通过

**核心价值**：
- 消除"活动选择困扰"
- 确保活动与发展阶段匹配

---

### 3. 每日记录 — 基础观察数据采集
**功能描述**：
- 快速记录日常观察（孩子行为、活动执行、发展变化）
- 支持文字、照片、标签记录
- 与时光胶囊联动，累积成长数据

**实装状态**：✅ 已完成
- 前端: diary.js（日记功能）+ today.js（今日视图）
- 后端: REST API for daily records
- 测试: 52/244 测试通过

**核心价值**：
- 降低记录成本（快速、轻量）
- 为复盘分析提供数据源

---

### 4. 时光胶囊 — 解决"成长不可见"痛点（视觉维度）
**功能描述**：
- 汇总孩子的成长瞬间（照片、视频、笔记）
- 生成按月龄分段的成长回顾
- 支持时间轴浏览和对比查看（如：3 个月时 vs 6 个月时）
- 导出成长档案（PDF/相册）

**实装状态**：✅ 已完成
- 前端: moments.js（时光页面重构）
- 后端: Aggregation API for timeline view
- 测试: 48/244 测试通过

**核心价值**：
- 让家长「看见」孩子的连续成长
- 增强记录的情感反馈

---

### 5. 复盘分析 — 解决"成长不可见"痛点（数据维度）
**功能描述**：
- 基于 216 个发展里程碑（10 个维度）自动评估孩子进度
- 生成发展报告（每个维度的当前阶段、预期目标、建议）
- 反思家庭教学策略效果
- 调整下阶段活动计划

**维度包括**：
1. 大肌肉发展（爬、行走、跑）
2. 精细肌肉发展（拿取、握笔、剪纸）
3. 语言理解与表达
4. 认知与逻辑（数字、分类、因果）
5. 社交能力（交互、合作、情绪）
6. 独立性与自理（穿衣、进食、如厕）
7. 创意与想象力
8. 音乐与节奏感
9. 观察与专注力
10. 适应与学习力

**实装状态**：✅ 已完成
- 前端: milestones.js（里程碑进度追踪）+ analysis dashboard
- 后端: milestones-complete.js（216 个里程碑定义）+ scoring API
- 测试: 32/244 测试通过

**核心价值**：
- 数据化呈现孩子发展现状
- 帮助家长评估教学有效性
- 科学指导下一阶段教学调整

## Should Have 清单（第二期候选）

以下功能在产品规划中但不在 MVP 范围内：

| 功能 | 原因 | 优先级 |
|------|------|-------|
| 家长社区 | 依赖用户基数达到临界量 | P2 |
| 专家咨询 | 需要蒙台梭利认证讲师资源 | P2 |
| 分享与协作 | MVP 仅支持单一主人，多人协作为后期功能 | P2 |
| 数据可视化图表 | 当前用 milestone dashboard 替代，计划优化图表质量 | P2 |
| 离线完全支持 | 当前支持草稿离线编辑，完全离线为优化项 | P3 |

---

## Won't Have 清单（明确不做）

| 功能 | 原因 |
|------|------|
| 商务模式（付费功能/广告） | MVP 专注产品验证，不做变现 |
| 第三方集成（微信/钉钉/飞书） | 家庭教育场景暂不需要企业协作 |
| AI 自动分析 | 当前依靠人工规则库，AI 功能为远期探索 |
| 多语言国际化 | MVP 专注中文市场 |
| 硬件集成（智能家居/穿戴设备） | 家庭蒙台梭利不依赖硬件 |

## 功能与痛点映射

| 核心痛点 | 解决功能 | 验证方式 |
|---------|--------|--------|
| 观察无体系 | 成长档案 + 活动推荐（蒙梭模块框架） | docs/03 验收标准：能否按模块查看观察 |
| 活动库不适龄 | 活动推荐（143 个适龄活动） | docs/03 验收标准：推荐活动与月龄匹配度 |
| 成长不可见 | 时光胶囊（视觉）+ 复盘分析（数据） | docs/03 验收标准：能否生成成长报告 |
